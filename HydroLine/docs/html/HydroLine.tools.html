HydroLine.tools.outil_points_bas module¶

HydroLine.tools.outil_points_bas.select_next_pixel_bas(courant, candidats_voisins, elevation_courante, arrivee_px, resoudre_egalite)¶

Sélectionne le prochain pixel en suivant les points bas parmi les candidats voisins.

Cette fonction évalue les pixels voisins d’un pixel courant pour déterminer le prochain pixel dans le chemin en fonction des élévations et la destination souhaitée.

Paramètres:

  • courant (tuple) – Coordonnées (cx, cy) du pixel courant.

  • candidats_voisins (list of dict) – Liste des voisins candidats avec leurs informations, incluant “elevation”.

  • elevation_courante (float) – Élévation du pixel courant.

  • arrivee_px (tuple) – Coordonnées (cx, cy) du pixel d’arrivée désigné.

  • resoudre_egalite (function) – Fonction pour départager les candidats en cas d’égalité d’élévations.

Renvoie:

Coordonnées (cx, cy) du prochain pixel à sélectionner.

Type renvoyé:

tuple
